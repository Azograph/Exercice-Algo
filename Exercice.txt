• Exercice 1 
Ecrire l'algorithme suivant en pseudo code
Stocker l’année de naissance
Stocker l’année actuelle
Calculer l’âge
Afficher le résultat

1. Créer une variable "AnnéeNaissance"
2. Demander AnnéeNaissance
3. AnnéeNaissance <- AnnéeNaissance entrée
4. Créer une variable "AnnéeActuelle"
5. Demander AnnéeActuelle
6. AnnéeActuelle <- AnnéeActuelle entrée
7. Crée "Age" 
8. Age <- AnnéeActuelle - AnnéeNaissance
9. Afficher Age
